What is A-Frame?

An open-source, web framework (JavaScript library) for building virtual reality experiences.

Compatible with: Desktop Browsers / Mobile Browser / HTC Vive / Oculus Rift / Google Daydream / Samsung GearVR / Cardboard

Features of A-Frame

The Mozilla VR team developed A-Frame to be the easiest and most powerful way to create WebVR content.

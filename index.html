What is A-Frame?

An open-source, web framework (JavaScript library) for building virtual reality experiences.

Compatible with: Desktop Browsers / Mobile Browser / HTC Vive / Oculus Rift / Google Daydream / Samsung GearVR / Cardboard

Features of A-Frame

The Mozilla VR team developed A-Frame to be the easiest and most powerful way to create WebVR content.

Applications of A-Frame

Showcase captured content with AFrame. It is viewable 360° and VR-ready images and integrate with other web frameworks.

Applications of A-Frame

Interactive Environments

AFrame can be used to build completely immersive VR environments on the web, that include movement, animations, and interactivity.

Applications of A-Frame

Multi-User Experiences

Make your virtual reality environments social with “Networked AFrame”, a web framework for multi-user experiences. Different users are able to access the same A-Frame scene.

Link: https://goo.gl/nd8Me1

Applications of A-Frame

Link Traversal

Fulfill the “Web” in WebVR. Travel from world to world and stay in VR as you travel between VR web pages (currently in experimental builds).

Applications of A-Frame

Augmented Reality
